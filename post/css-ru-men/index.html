CSS入门

2022-09-15

概念：Cascading Style Sheet 层叠样式表

  • 层叠：多个样式可以作用在同一个html的元素上，同时生效

好处

  1. 功能强大
  2. 将内容展示和样式控制分离
    • 降低耦合度，解耦
    • 让分工协作更容易
    • 提高开发效率

CSS 与 HTML

  1. 内联样式
    • 在标签内使用style属性指定css代码
      * 如**[div style="color:red;"]hello css[/div]**
  2. 内部样式
    • 在head标签内定义style标签，style标签的标签体的内容就是css代码
    • 如：
<style>
          div{
              color:blue;
          }
  <style>
  1. 外部样式
    1. 定义css文件
    2. 在head标签里定义link标签，引入外部资源
    3. [link rel="stylesheet" href="css/a.css"]
  • 注意：
    • 1,2,3种方式作用域越来越大
    • 1不常用，后期常用2,3

    CSS基本语法

    1. 格式：
             选择器{
             属性名1：属性值1；
             属性名2：属性值2；
             · · · · · ·
         }
    1. 选择器：筛选具有相似特征的元素
    2. 注意：每一对属性需要用；隔开，最后一堆属性可以不加；

选择器

  • 概念：筛选具有相似特征的元素
  • 分类
    • 基础选择器
      • id选择器：选择器具体的id属性值的元素，建议在一个html页面中id值唯一
        • 语法：#id属性值{}(优先级最高)
      • 元素选择器：选择具有相同标签名称的元素
        • 语法：标签名称{}
        • 注意：id选择器优先级高于元素选择器
      • 类选择器：选择具有相同class属性值的元素
        • 语法：.classs属性值{}
        • 注意：类选择器优先级高于元素选择器
        • [p class="cls1"]blah[/p]
      • #id{} >>> .class>>>标签名称
    • 扩展选择器：
      • 选择所有元素：
        • 语法：*{}
      • 并集选择器：
        • 选择器1，选择器2{}
      • 子选择器：筛选选择器1内的选择器2
        • 语法：选择器1 选择器2{}
      • 父选择器：筛选选择器2的父选择器1
        • 语法：选择器1 > 选择器2{}
      • 属性选择器：选择元素名称，属性名=属性值的元素
        • 语法： 元素名称【属性名="属性值"】{}
        • input[type="text"]{}
      • 伪类选择器：选择一些元素具有的状态
        • 语法：元素：状态{}
        • 如：
          • 状态：
            • link：初始化的状态 a：link{color:pink}
            • visited：被访问过的状态
            • active：正在被访问的状态
            • hover：鼠标悬浮的状态

    属性

    1. 字体、文本
      • font-size:字体大小
      • color：颜色
      • text-align：对齐方式
      • line-height：行高
    2. 背景
      • background
    3. 边框
      • border：边框（复合属性）
    4. 尺寸
      • width
      • height
    5. 盒子尺寸
      • margin:外边距
      • padding：内边距
        • 默认情况下内边距会影响整个盒子的大小
        • box-sizing：border-box；设置盒子的大小，让width和height就是盒子的最终大小
        • float：浮动
        • left
        • right
div{
    border: 1px solid red;
    height: 200px;
    width: 200px;
    background: url("img/logo.jpg") no-repeat center;
}
