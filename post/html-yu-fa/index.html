html基础语法

2022-09-13

概念

  • 超文本标记语言
    • 超文本
      • 超文本是用超链接的方法，将各种不同空间的文字信息组织在一起的网状文本
    • 标记语言
      • 标记语言不是编程语言
      • 由标签构成的语言。<标签名称>如 html，xml

    语法

    • html文档后缀名.html 或者 .htm
    • 标签分为
      1. 围堵标签：有开始标签和结束标签。如
      2. 自闭合标签：
    • 在开始标签中可以定义属性。属性是由键值对构成，值需要引号（单双皆可）引起来
    • html不区分大小写，但推荐小写。

    标签

1. 文件标签：构成html最基本的标签

    * html：html的根标签
    * head：头标签，引入资源
    * title：标题标签
    * body： 体标签

2. 文本标签：和文本有关的标签

    * 注释`<!--blah-->`
    * h1 to h6 标题
    * p 段落
    * br 换行
    * hr 水平线
        * 属性
            * color
            * width
            * size
            * align
                * center
                * left
                * right
    * b 加粗
    * i 斜体
    * font 字体（html5已淘汰）
        * 属性
            * color 颜色
                1. 英文单词：red，green，blue
                2. rgb(a,b,c) 
                3. #abc: $a,b,c \in 0x00 to 0xFF$
            * size 大小
            * face 字体
            * width:
                * 数值：width=‘20’，数值单位默认是px（像素）
                * 数值%：占比相对于父元素的大小

3. 图片标签：

<img src = "image/jingxuan_2.jpg" align = "right" alt = "alternative" />
  • src: 指定图片位置
    • 以.开头的路径
      • ./: 表示当前目录
      • ../: 表示上一级目录

4. 列表标签：

1. ol+li
2. ul+li

5. 连接标签：

1. a: 定义一个超链接
    1. 属性：
        1. href：制定访问资源的URL
        2. target：制定打开资源的方式
            1. _self:默认值,在当前页面打开
            2. _blank:在空白页面打开

6. div和span：

  • span：文本信息在一行内展示，行内标签 内联标签
  • div：每个div占满一整行。块级标签

7. 语义化标签： 为了提高程序可读性，提供了一些标签（自身没有任何形式,结合CSS）

1. header
2. footer

8. 表格标签：

* table: 定义表格
    * width： 宽度
    * border：边框
    * cellpadding：定义内容和单元格之间的距离
    * cellspacing: 定义单元格之间的距离。若指定为0，则边框合为一条线
    * align： 对齐方式
* tr：行
* td：列
    * colspan: 合并列 `<th rowspan="2">`
    * rowspan: 合并行
* th：表头单元格
* caption: 标题 
* thead: 表示表格的头部分
* tbody: 表示表格的体部分
* tfoot: 表示表格的尾部分
